# Que es iWatch?

```
iWatch es una plataforma de entretenimiento online,
donde podrás disfrutar de una amplia gama de películas 
y series, a través de una interfaz intuitiva y apta para 
todos los públicos. 

También podrás compartir tú opinión con el resto de usuarios, 
además de acceder a una comunidad llena de listas de reproducción 
con las que aprender y disfrutar de todo el contenido de la plataforma.
```

## Como usamos la aplicación?

```

Previamente al uso y desarrollo del proyecto, se requiere la instalación y 
puesta en marcha de ciertas herramientas. Para ello, sigue las instrucciones
paso por paso que aparecen a continuación.

```

### Clonar Repositorio del Github

```bash
git clone ....
```

### Proporcionar información sobre los repositorios asociados al proyecto

```bash
git remote -v
```

### Mostrar información sobre el estado del repositorio

```bash
git status
```

### Mostrar información sobre las ramas

```bash
git branch -a
```

### Conectar repositiorio local a uno remoto

```bash
git remote add origin URL_DEL_REPORITORIO
```

### Subir cambios locales

```bash
git push origin main
```

### Instalación vía Composer

```bash
composer install
```

### IMPORTANTE - Copiar el fichero .env.example  a .env, edita las credenciales y la url

```bash
composer install
```

### Generar Application Key

```bash
php artisan key:generate
```

### Migrar base de datos

```bash
php artisan migrate
```

### Borrar y recrear tablas + sembrado de datos en la base de datos

```bash
php artisan migrate:fresh --seed
```

### Lanzar Seeders

```bash
php artisan db:seed
```

### Instalar las dependencias de Node

```bash
npm install

npm run dev
```
### Lanzar a producción

```bash
npm run build or yarn build
```
### Inicialización del servidor

```bash
php artisan serve
```
