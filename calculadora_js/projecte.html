Què és RSS?

Es tracta d'un tipus de format de dades que s'utilitza principalment per a difondre contingut als subscriptors d'un lloc web.

Per a què s’utilitza RSS?

L'RSS permet distribuir contingut sense necessitat d'utilitzar un navegador web mitjançant l'ús d'un software dissenyat per a llegir els continguts RSS.

Com saber si una pàgina utilitza RSS?

Per a comprovar si una pàgina utilitza RSS, només haurem de fer clic dret sobre l'adreça de la web i seleccionar l'opció de Codi font.

Posteriorment, dins de de la finestra que s'obrirà haurem de buscar la paraula "rss" mitjançant la comanda Ctrl + F o Command + F.

Si la pàgina disposa d'RSS, trobarem el feed entre cometes després d'un "href=".

Què és xQuery?

Es tracta d'un llenguatge de consulta que permet extreure informació d'una base de dades o documents XML.

xQuery es basa en el llenguatge xPath per a l'accés als nodes XML.
